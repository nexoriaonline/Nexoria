# 🚀 Nexoria Online - Progresso do Projeto

## ✅ Concluído
- **Servidor TFS (8.60):** Compilado e configurado com banco de dados MariaDB.
- **Protocolo de Rede:**
  - RSA desabilitado para compatibilidade com Web Browser.
  - Ajuste de cabeçalho de bytes (Protocol ID 0x01/0x0A).
  - Suporte a Adler32 Checksum implementado no Cliente e Servidor.
  - Fluxo de Login completo: Login Server -> Challenge -> Game Server.
- **Cliente Web:**
  - Renderização via Canvas/WebGL.
  - Joystick virtual para mobile e WASD para PC.
  - Sistema de Chat, Minimapa e Inventário.
  - Movimento e animações básicas.
- **Infraestrutura:**
  - Proxy WebSocket (Websockify) configurado para portas 8001 (Login) e 8002 (Game).
  - Repositório GitHub atualizado com todas as correções.

## 🚧 Em Andamento
- **Conteúdo do Jogo:** Adição de mais monstros e NPCs.
- **Balanceamento:** Ajuste de dano e experiência.
